# Memory Game
This is a game built with HTML, CSS, JavaScript, and jQuery. It tests how you can memorize the pattern of blinking of some randomly blinking boxes. It can help you improve your retentive memory.
## Built With

- HTML
- CSS
- JavaScript
- jQuery
